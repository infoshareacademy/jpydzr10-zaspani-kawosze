# jpydzr10-zaspani-kawosze

Prosty prototyp aplikacji CLI dla siłowni.

## Uruchomienie

1. Zainstaluj zależności:
   `python3 -m pip install -r requirements.txt`
2. Uruchom aplikację:
   `python3 menu.py`

## Obecny zakres

- menu tekstowe,
- podgląd cennika,
- placeholder grafiku i pozostałych sekcji.

Jeśli plik `grafik.csv` nie istnieje, aplikacja wyświetli komunikat zamiast błędu.
